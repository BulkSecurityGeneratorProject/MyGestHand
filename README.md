MyGestHand
==========

Web Interface to manage handball club

Features
--------
  - Manage club :
    - Manage leadership
    - Manage coatchs
  - Manage all contacts of the club
    - Individual contact informations
    - Licencies for each member of the club
    - publipostage
    - mailing list
  -  Manage team :
    - Players of the team
    - Players statistics
    - Match calendar
